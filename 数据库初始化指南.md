# MySQL数据库初始化指南

## 当前状态
✅ MySQL80服务正在运行

## 初始化步骤

### 1. 打开MySQL客户端
您可以使用以下任一方式：
- MySQL Workbench (推荐)
- MySQL命令行客户端
- phpMyAdmin
- 其他MySQL管理工具

### 2. 连接到MySQL
- 主机: localhost
- 端口: 3306
- 用户名: root
- 密码: [您的MySQL root密码]

### 3. 执行初始化脚本
复制并执行 `BackEnd/src/main/resources/init-database.sql` 文件中的所有SQL语句，或者直接导入该文件。

### 4. 验证数据库创建
执行以下SQL验证数据库是否创建成功：

```sql
-- 检查数据库
SHOW DATABASES LIKE 'fuzz_testing_db';

-- 使用数据库
USE fuzz_testing_db;

-- 检查表
SHOW TABLES;

-- 检查数据
SELECT COUNT(*) FROM database_config;
SELECT COUNT(*) FROM parameter_template;
```

### 5. 预期结果
- 数据库 `fuzz_testing_db` 已创建
- 表 `database_config` 包含 2 条记录
- 表 `parameter_template` 包含 15 条记录

## 后端配置
当前后端配置为：
- 数据库URL: `jdbc:mysql://localhost:3306/fuzz_testing_db`
- 用户名: `root`
- 密码: `password`

如果您的MySQL密码不是 `password`，请修改 `BackEnd/src/main/resources/application.yml` 文件中的密码配置。
